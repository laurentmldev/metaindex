 ## SSL certificate process
 ## https://stackoverflow.com/questions/21141215/creating-a-p12-file
 Generate a private key on target server
 $ openssl genrsa -out myKey.pem
 
 Generate a certificate request file (csr)
 $ openssl req -new -key myKey.pem -out cert.csr

 Provide it to certification authority
 
 Retrieve corresponding certificate file
 
 Create  (public key) certificate pem file from pkcs7 if such format is provided by CA
 $ openssl pkcs7 -in myCert.cer -print_certs -out certs.pem
 
 Build p12 file
 $ openssl pkcs12 -export -out keyStore.p12 -inkey myKey.pem -in certs.pem
